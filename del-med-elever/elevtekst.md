# ⚔️ Bli med i Vikingenes kulturmøter!

Åpne spillet på mobil, iPad eller PC:

**https://vikingspill.vercel.app/**

Slik kommer dere i gang:

1. Trykk på lenken (eller skann QR-koden).
2. Velg **«Jeg er elev»**.
3. Tast inn **spillkoden** læreren viser på storskjermen.
4. Velg vikingskip, gi det navn og symbol — og sett seil! 🛶

God reise, vikinger!
